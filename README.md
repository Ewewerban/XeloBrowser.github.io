### 🟧 About the Project

XeloBrowser was created with ultimate simplicity and a unique look in mind. The application loads a dedicated black start page with orange border accents, allowing for instant web searching via a secure redirection to Google.

### 🟧 Key Features

```diff
+ 🚀 Chromium Core (WebView2) – stable and lightning-fast rendering of modern websites.
+ 🟧 Unique Design – user interface based on black, white, and orange borders.
+ 🔍 Google in a Single Tab – smart redirection inside the application (does not open external windows).
+ 💾 Full Cache Persistence – your custom shortcuts (tiles) and search history do not disappear after closing the program.
