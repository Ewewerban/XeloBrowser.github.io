# VeloBrowser 🚀

VeloBrowser to nowoczesna, minimalistyczna przeglądarka internetowa napisana w języku Python. Bazuje na silniku Chromium (tym samym co Google Chrome), a jej domyślną stroną startową jest Google.

## Funkcje
- Szybkie ładowanie stron dzięki Chromium
- Klasyczna nawigacja: Wstecz, Dalej, Odśwież, Strona Główna (Google)
- Pasek postępu ładowania strony
- Inteligentny pasek adresu

2. Pobierz te pliki z repozytorium.
