### 🟧 O projekcie

XeloBrowser został stworzony z myślą o maksymalnej prostocie i unikalnym wyglądzie. Aplikacja ładuje dedykowaną, czarną stronę startową z pomarańczowymi akcentami ramkowymi, która pozwala na natychmiastowe wyszukiwanie w sieci za pomocą bezpiecznego przekierowania do Google.
