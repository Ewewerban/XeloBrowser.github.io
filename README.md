# XeloBrowser 🚀

VeloBrowser to nowoczesna, minimalistyczna przeglądarka internetowa. Bazuje na silniku Chromium (tym samym co Google Chrome), a jej domyślną stroną startową jest Google.

## Funkcje
- Szybkie ładowanie stron dzięki Chromium
- Klasyczna nawigacja: Wstecz, Dalej, Odśwież, Strona Główna 
- Pasek postępu ładowania strony
- Inteligentny pasek adresu
