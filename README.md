# XeloBrowser 🚀

XeloBrowser to nowoczesna, minimalistyczna przeglądarka internetowa systemu Windows oparta na wydajnym silniku Chromium (WebView2). Domyślną stroną główną aplikacji jest autorski, minimalistyczny czarno-pomarańczowy pulpit nawigacyjny zintegrowany z bezpośrednim przekierowaniem do wyszukiwarki Google.

## Funkcje projektu

* **Rdzeń Chromium:** Pełna szybkość renderowania nowoczesnych stron dzięki bibliotekom WebView2.
* **Klasyczny układ przycisków:** Wstecz, Dalej, Odśwież oraz powrót na stronę domową.
* **Czarno-Pomarańczowy Interfejs:** Wyrazisty i czytelny wygląd oparty wyłącznie na czerni, bieli i pomarańczowych ramkach akcentowych.
* **Bezpośrednie wyniki Google:** Pasek wyszukiwania natychmiastowo przekierowuje kartę do oficjalnej wyszukiwarki Google.
* **Moduł Cookies i Pamięci lokalnej:** Zachowywanie dodanych skrótów (kafelków) i historii wyszukiwania pomiędzy sesjami działania przeglądarki.
